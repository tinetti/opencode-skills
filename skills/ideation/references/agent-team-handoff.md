# Agent Team Handoff

Background context for the execution handoff when parallel phases are available.

## Agent Teams

- Use a single lead session to coordinate teammates.
- The lead should spawn teammates, assign work, approve plans, and synthesize results.
- Prefer fresh sessions for each phase.

## Why fresh sessions?

- Ideation consumes significant context.
- Execution benefits from clean, phase-focused context.
- Human review between phases catches issues early.
- Each phase should be independently committable.
