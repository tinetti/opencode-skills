# HTML Guide

Use this before generating any HTML artifact for ideation.

## Rules

- One self-contained `.html` file.
- Inline all CSS and JS.
- No CDN links, no build step, no remote fonts.
- Must work from `file://` offline.
- Support both light and dark mode.
- Use semantic HTML.
- Keep JS minimal.
- Use design tokens rather than hardcoded styles.

## Common structure

`<html>` -> `<header class="doc-header">` -> `<main class="doc-content">`.

Use this guide for contract HTML and any temporary comparison artifacts.
